1. Why Python is easy compared to C?

Python is easier because:

Simple syntax

Less code

No datatype declaration

No semicolon ;

No header files

Built-in functions available


Example:

C

#include<stdio.h>

int main()
{
    int a = 10;
    printf("%d",a);
}

Python

a = 10
print(a)

Python looks cleaner and closer to English language.


---

2. Why no datatype declaration needed?

Python is dynamically typed language.

It automatically detects datatype.

Example:

a = 10

Python understands:

a is integer


Example:

name = "Ram"

Python automatically knows:

name is string


Unlike C:

int a;
char name[20];


---

3. Why no semicolon in Python?

Python uses:

newline

indentation


to identify statement ending.

So semicolon is usually unnecessary.

Example:

a = 10
print(a)

Each line itself is separate statement.

In C:

a = 10;
printf("%d",a);

; tells statement end.


---

4. Why indentation important?

Python uses indentation instead of {}.

Indentation defines block of code.

Example:

if a > 10:
    print("Big")

The spaces show:

print() belongs to if block


Without indentation:

if a > 10:
print("Big")

Python gives error.

In C:

if(a > 10)
{
    printf("Big");
}

C uses {}. Python uses spaces.

5. Difference between list and array?

List	Array

Built-in Python collection	Usually fixed-size structure
Can store different datatypes	Usually same datatype
Dynamic size	Fixed size
Uses []	Array module/library needed


Example list:

lst = [1, "Ram", 5.5]

Different datatypes allowed.

In C array:

int arr[5];

Only integers.

6. What is interpreter?

Interpreter converts program:

line by line

into machine language


Python uses interpreter.

Meaning:

code executes one line at a time



---

Compiler vs Interpreter

Compiler	Interpreter

Whole program at once	Line by line
Faster execution	Slower
Example: C	Example: Python

7. What is dynamic typing?

Dynamic typing means:

datatype assigned automatically during execution


Example:

a = 10

a becomes integer.

Later:

a = "Hello"

Now a becomes string.

Same variable can store different datatypes.

This is dynamic typing.
